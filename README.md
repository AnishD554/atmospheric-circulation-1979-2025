ERA5 Atmospheric Circulation Analysis (1979–2025)

This repository contains Python notebooks used to analyze multidecadal changes in
large-scale atmospheric circulation using ERA5 reanalysis data.

Analyses include:
- Hadley, Ferrel, and Polar cell boundary trends
- Arctic stratospheric variability diagnostics
- ENSO phase-space embedding using tropical Pacific SST data

All figures in the project poster were generated using these scripts.
